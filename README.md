# Emoto Support Platform

這個 repo 是依據 `Emoto` 在全國大專校院 Healthy x Happy 創新創業競賽中的得獎作品概念做出的前端原型。原型把「情緒零時差：Emoto 線上心理支持平台」實作成一個情緒登記、匿名支持媒合、陪伴時段與危機分流工作台。

## 比賽來源

- 比賽/計畫：全國大專校院 Healthy x Happy 創新創業競賽
- 年度：114 年
- 屆次/階段：2025 第九屆決賽獲獎名單
- 獎項：佳作獎
- 團隊名稱：Emoto
- 得獎作品：情緒零時差：Emoto 線上心理支持平台
- 主辦：國立臺北護理健康大學育成中心
- 官方來源：https://incubation.ntunhs.edu.tw/tab/528/id/1834

## 核心概念

許多人在情緒低潮時需要即時、低門檻、不中斷的陪伴，但不一定能立刻找到可信任的人或專業資源。這個原型把概念拆成五個可軟體化流程：

- 情緒登記：使用者用簡短量表與文字紀錄當下狀態。
- 匿名支持媒合：依情緒主題、等待時間與支持者專長媒合陪伴時段。
- 支持任務台：支持者查看待回應需求、回覆節奏與陪伴腳本。
- 危機分流：高風險訊號會提示轉介緊急資源或專業協助。
- 成效追蹤：追蹤情緒變化、回訪提醒與支持完成率。

## 原型範圍

這是靜態前端原型，資料皆為模擬。它不是醫療診斷或治療工具，不代表原團隊正式產品，也未使用原團隊未公開資料、商標素材或後端服務。

## 使用方式

用瀏覽器開啟 `index.html`，或透過主整理網站的原型連結進入。
